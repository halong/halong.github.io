使用hexo生成博客网站

hexo是什么？

Hexo是一个基于node.js可以将Markdown文件快速的生成静态网页的博客框架.

安装git

下载安装git。在控制台输入: git –version,检查是否成功

安装node.js

下载安装node.js。在控制台输入: npm -v,检查是否成功

安装hexo

  • 新建一个文件夹hexo，在该文件夹内右击打开git bash，执行：npm install -g hexo 安装hexo
  • 执行：hexo init 初始化hexo
  • 执行：hexo s 开启服务，浏览器输入：http://localhost:4000/ 即可访问,类似于tomcat.

在github申请pages

只需要创建仓库的名字格式为：yourname.github.io 就会自动创建
